# Course Title

Modern AI Tech Stack in 3 Days: APIs, Open-Source Models, RAG and Responsible AI

## Proposed By

Darshan

## Course Overview

This 3-day program provides a structured immersion into the modern AI technology stack. Participants move from API-based AI integration to open-source models, embeddings, vector search, and Retrieval-Augmented Generation (RAG). The program focuses on building practical AI workflows and understanding how enterprise-ready AI systems are designed, evaluated, and governed responsibly.

## Course Outcomes

By the end of the program, participants will be able to:

1. Integrate AI services through REST APIs for text, image, and multimodal workflows.
2. Use open-source models from Hugging Face for common NLP and vision tasks.
3. Understand embeddings and implement semantic search using vector databases.
4. Build a document-based question answering system using LangChain and RAG.
5. Evaluate AI responses and apply responsible AI principles in system design.

## Prerequisites

* Basic programming knowledge (preferably Python)
* Familiarity with JSON and CSV data formats
* Laptop with Google Colab or Jupyter enabled
* Interest in applied AI systems and open-source tools
